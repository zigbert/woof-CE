Belenos Puppy DISTRO_VERSION - released December 2015

Belenos Puppy DISTRO_VERSION

December, 2015

The debut release of Belenos Puppy Linux is out!

Belenos Puppy, or belpup, is built from Trisquel belenos DEB packages, hence has binary compatibility with Trisquel and access to the Trisquel 7 Belenos repositories. .

Features include:

  • Updated "First Run" GUI with bugfixes and visual enhancements.
  • New kernels deblobbed with Linux-libre, following LTS branchess. The shipped kernel is 3.14.x.
  • New "change_kernels" script to easily change kernels
  • Many Icon and Graphical User Interface improvements throughout the system.
  • QupZilla open source and free (as in freedom) browser
  • new graphical firewall program
  • Many other updated programs.

As mentioned above, belpup DISTRO_VERSION is shipped with kernel 3.14.x, however kernels can be compiled from kernel-kit and easily changed using the change_kernels script. For more information see the puppy forum page.
